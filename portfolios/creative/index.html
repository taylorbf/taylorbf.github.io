Ben Taylor: Select Performances

> Dreams of Massachusetts

Live coding a robotic zither designed by EMMI (Expressive Machines, Musical Instruments). This performance uses a homemade live coding language I designed to explore the use of audio signals as pattern generators. Performed at the 2017 New Interfaces for Musical Expression (NIME) conference in Copenhagen, DK.

> The Last Cloud

A media art performance in which I look back on my daily life using the main technology of my daily life: the web browser. In The Last Cloud I live code the browser in a collage of internet media, cell phone videos, and websites. Glitching quicktime players; harmonizing with dialtones; browsing as performance; browsing as dreaming.

> WonderGarden

A music performance that uses the audience's phones as a collective speaker system. A performer uses the internet to send control gestures to web browsers in audience phones, causing them to synthesize audio. Performed at Mill 180 Park in Easthampton, MA with a grant from Easthampton City Arts. For more on this genre, see my paper A History of the Audience as a Speaker Array.
