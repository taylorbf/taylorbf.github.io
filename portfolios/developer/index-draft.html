Grapph

Grapph is an app that helps users chart progress over time. The app offers an elegantly designed interface to chart any type of ongoing data, whether that be weight, work hours, TV hours, or something else. The simple UI lets users quickly open the app, add an entry, and leave. I was the sole graphic designer and front-end developer for this project.

For Create1, I developed a working prototype of their browser-based remixer. NexusUI

NexusUI is an open-source JavaScript toolkit of musical interface components. Developers can use NexusUI to rapidly create audio projects in the browser using NexusUI and the Web Audio API. NexusUI is my most popular project, being featured on Hacker News in 2014 and being used by thousands of developers around the world. Today NexusUI sees about 50 unique users each day. I have been the primary developer and maintainer of the project since 2013.
